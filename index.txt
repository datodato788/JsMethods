1.length: მასივის ელემენტების რაოდენობა.
2.push(): ამატებს ელემენტს მასივის ბოლოს.
3.pop(): შლის ბოლო ელემენტს მასივიდან.
4.shift() შლის პირველ ელემენტს მასივიდან
5.unshift() ამატებს ახალ ელემენტებს მასივის დასაწყისში და ბრუნდება ახალი სიგრძე.
6.concat() ქმნის ახალ მასივს, რომელიც შედგება არსებული და დამატებული ელემენტებისგან.
7.replace() ცვლის სტრიქონის ნაწილს ახალი მნიშვნელობით.
8.startsWith() ბრუნდება true თუ სტრიქონი იწყება მოცემული სიმბოლოებით, false თუ არა.
9.endsWith()ბრუნდება true თუ სტრიქონი მთავრდება მოცემული სიმბოლოებით, false თუ არა.
10.charCodeAt() ბრუნდება ASCII კოდი მოცემული ინდექსის სიმბოლოს.
11.repeat() ბრუნდება ახალი სტრიქონი, რომელიც შედგება არსებული სტრიქონის გამეორებით მოცემული რაოდენობის.
12.search() ეძებს სტრიქონში მოცემული რეგულარული გამომეტყველებით და ბრუნდება ინდექსი, სადაც იწყება ნაპოვნი ტექსტი.
13.match() ეძებს სტრიქონში მოცემული რეგულარული გამომეტყველებით და ბრუნდება ნაპოვნი ტექსტი, ან null თუ არ იპოვება.
14.every() ბრუნდება true თუ ყველა ელემენტი აკმაყოფილებს მოცემულ ფუნქციას, false თუ არა.
15.some() ბრუნდება true თუ რომელიმე ელემენტი აკმაყოფილებს მოცემულ ფუნქციას, false თუ არა.
16.flat() ქმნის ახალ მასივს, რომელიც არის მოცემული მასივის გაბრტყელებული ვერსია (ერთ დონემდე).
17.flatMap() ქმნის ახალ მასივს, რომელიც არის მოცემული ფუნქციის გამოყენების შემდეგ გაბრტყელებული შედეგებისგან.
18.padStart() ასრულებს სტრიქონს მოცემული სიმბოლოებით სანამ არ მიაღწევს განსაზღვრულ სიგრძეს დასაწყისიდან.
19.padEnd() ასრულებს სტრიქონს მოცემული სიმბოლოებით სანამ არ მიაღწევს განსაზღვრულ სიგრძეს ბოლოში.
20.startsWith() აბრუნებს true თუ სტრიქონი იწყება მოცემული სიმბოლოებით, false თუ არა.
